[Solution] BSOD 0xc000014c STATUS_BOOT_CRITICAL Windows 11/10 — Fixed

Error 0xc000014c is a critical BSOD STOP error with the status code STATUS_BOOT_CRITICAL. It occurs when Windows encounters a fatal error during the boot process that prevents the system from reaching the operating system. The boot loader or kernel cannot access critical boot files, and the system halts immediately.

This error affects both Windows 10 and 11 and almost always appears during boot, resulting in a recovery screen or a system that cannot start at all.

Common Causes

  • Corrupted Boot Configuration Data (BCD) — The BCD store is damaged or points to missing boot files.
  • Disk corruption — Bad sectors on the system drive prevent boot files from loading.
  • Failed Windows Update — An interrupted update corrupts boot-related files.
  • File system corruption — NTFS or FAT corruption on the boot volume makes boot files unreadable.

How to Fix

Rebuild BCD

Boot from a Windows installation USB or recovery drive:

  1. Select Repair your computer.
  2. Go to Troubleshoot > Advanced options > Command Prompt.
  3. Run:
bootrec /fixmbr
bootrec /fixboot
bootrec /rebuildbcd

If bootrec /rebuildbcd finds no installations:

bcdboot C:\Windows /s C: /f ALL

If bootrec /fixboot fails:

bootsect /nt60 sys
bootrec /fixboot

Run CHKDSK from Recovery

Disk corruption is a primary cause of this error:

chkdsk C: /f /r /x

Replace C: with the correct drive letter if Windows is on a different partition. The /r flag locates and recovers data from bad sectors.

Run SFC from Recovery

sfc /scannow /offbootdir=C:\ /offwindir=C:\Windows

Run DISM from Recovery

DISM /Image:C:\ /Cleanup-Image /RestoreHealth

Verify Boot Files Exist

Check that the critical boot files are present:

dir C:\Windows\System32\winload.exe
dir C:\Windows\System32\ntoskrnl.exe
dir C:\Boot\BCD

If any of these files are missing, the boot configuration is incomplete.

Check Disk Health

Get-PhysicalDisk | Select-Object FriendlyName, HealthStatus, OperationalStatus
Get-StorageReliabilityCounter -PhysicalDisk (Get-PhysicalDisk) | Select-Object Temperature, Wear, ReadErrorsTotal, WriteErrorsTotal

A failing drive should be replaced immediately. Back up data before replacement.

Perform System Restore

From the recovery environment:

rstrui.exe

Select a restore point dated before the error started.

Check and Repair MBR

bootrec /fixmbr
bootrec /fixboot

If the Master Boot Record is corrupted, these commands restore the standard Windows MBR.

Examples

This error commonly occurs in these scenarios:

  • After a forced shutdown during update — Interrupted writes corrupt the BCD or boot files.
  • With a failing hard drive — Bad sectors on the boot volume prevent critical files from loading.
  • After modifying partitions — Resize or delete operations on the boot partition corrupt the BCD.
  • Following a power outage — Unexpected power loss leaves the file system in an inconsistent state.
