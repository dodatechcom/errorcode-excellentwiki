TS2552 — Cannot find name ‘X’, did you mean ‘Y’

TS2552 is an enhanced version of TS2304 where TypeScript not only reports that a name isn’t found, but also suggests a similar name that might be what you intended. This makes fixing typos much easier.

Common Causes

// Cause 1: Typo in variable name
let userName = "Alice";
console.log(useName); // TS2552: Cannot find name 'useName'. Did you mean 'userName'?

// Cause 2: Typo in function name
function calculateTotal(items: number[]) {}
calculatTotal([1, 2, 3]); // TS2552: Did you mean 'calculateTotal'?

// Cause 3: Missing import with close alternative
import { UserService } from "./services";
const svc = new UsersService(); // TS2552: Did you mean 'UserService'?

// Cause 4: Wrong casing
const myArray = [1, 2, 3];
myarray.length; // TS2552: Did you mean 'myArray'?

Solutions

Fix 1: Use the suggested name

// Wrong
let userName = "Alice";
console.log(useName); // TS2552

// Correct — follow the suggestion
console.log(userName);

Fix 2: Import the correct name

// Wrong
import { UserService } from "./services";
const svc = new UsersService(); // TS2552

// Correct
const svc = new UserService();

Fix 3: Check for case sensitivity

// Wrong — lowercase 'r'
const result = myarray.reduce((a, b) => a + b, 0);

// Correct — PascalCase or camelCase as defined
const result = myArray.reduce((a, b) => a + b, 0);

Fix 4: Ensure the identifier is in scope

// Wrong — using variable before it's defined
console.log(myVar);
const myVar = 10; // TS2552

// Correct
const myVar = 10;
console.log(myVar);

Examples

// Example 1: DOM API typo
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContex("2d"); // TS2552: Did you mean 'getContext'?

// Example 2: Array method typo
const nums = [1, 2, 3, 4, 5];
const evens = nums.fliter(x => x % 2 === 0); // TS2552: Did you mean 'filter'?

// Example 3: Class method typo
class Calculator {
  add(a: number, b: number) { return a + b; }
}
const calc = new Calculator();
calc.addd(1, 2); // TS2552: Did you mean 'add'?
