[Solution] PHP Error 32767 — E_ALL Error Reporting Flag

PHP error code 32767 is the integer value of E_ALL. It represents the bitwise combination of all PHP error levels — E_ERROR, E_WARNING, E_PARSE, E_NOTICE, E_STRICT, E_DEPRECATED, E_USER_ERROR, E_USER_WARNING, E_USER_NOTICE, and E_USER_DEPRECATED. Setting error_reporting(E_ALL) ensures every type of PHP issue is surfaced.

Common Causes

  • Development environment not reporting all error types
  • Missing error_reporting configuration in php.ini
  • Production environments hiding notices and deprecations
  • Not distinguishing between development and production error levels

How to Fix

1. Enable Full Error Reporting in Development

<?php
error_reporting(E_ALL);
ini_set('display_errors', 1);
ini_set('display_startup_errors', 1);
ini_set('html_errors', 1);
?>

2. Configure php.ini for Development

; Development php.ini
error_reporting = E_ALL
display_errors = On
display_startup_errors = On
log_errors = On
error_log = /var/log/php_errors.log

3. Restrict Reporting in Production

<?php
// Production — report errors but don't display them
error_reporting(E_ALL & ~E_DEPRECATED & ~E_NOTICE);
ini_set('display_errors', 0);
ini_set('log_errors', 1);
ini_set('error_log', '/var/log/php_errors.log');
?>

4. Set Error Reporting Per Script

<?php
// At the top of a script for specific debugging
error_reporting(E_ALL);

// Or exclude specific levels
error_reporting(E_ALL & ~E_USER_NOTICE);
?>

All E_ALL Component Values

<?php
// Each error level and its integer value
echo 'E_ERROR: ' . E_ERROR . "\n";           // 1
echo 'E_WARNING: ' . E_WARNING . "\n";       // 2
echo 'E_PARSE: ' . E_PARSE . "\n";           // 4
echo 'E_NOTICE: ' . E_NOTICE . "\n";         // 8
echo 'E_STRICT: ' . E_STRICT . "\n";         // 2048
echo 'E_DEPRECATED: ' . E_DEPRECATED . "\n"; // 8192
echo 'E_USER_ERROR: ' . E_USER_ERROR . "\n";         // 256
echo 'E_USER_WARNING: ' . E_USER_WARNING . "\n";     // 512
echo 'E_USER_NOTICE: ' . E_USER_NOTICE . "\n";       // 1024
echo 'E_USER_DEPRECATED: ' . E_USER_DEPRECATED . "\n"; // 16384
echo 'E_ALL: ' . E_ALL . "\n";               // 32767
?>

Examples

<?php
// With E_ALL enabled, all of these will be reported:
echo $undefinedVar;           // E_NOTICE
$arr = [1, 2, 3];
echo $arr[10];               // E_NOTICE
$old_func = create_function('', 'return 1;'); // E_DEPRECATED
strlen(null);                // E_WARNING (PHP 8.0+)
?>
