Elixir Error Codes

Browse all Elixir error codes and solutions.

** (ArgumentError) wrong number of arguments

An ArgumentError occurs when calling a function with the wrong number of arguments.

** (BadMapError) expected a map

A BadMapError occurs when attempting to use a value as a map when it is not a map.

** (Enum.EmptyError) empty error

An Enum.EmptyError occurs when calling a function that requires at least one element on an empty enumerable.

** (FunctionClauseError) no function clause matching

A FunctionClauseError occurs when no function clause matches the given arguments.

** (FunctionClauseError) No Matching Function Clause

Elixir raises FunctionClauseError when no function clause matches the given arguments

** (MatchError) no match of right hand side value

A MatchError occurs when the right-hand side of a match operator (=) does not match the pattern on the left-hand side.

** (MatchError) no match of right hand side value

A MatchError occurs when a pattern match fails because the right-hand side value doesn't match the expected pattern.

** (NameError) undefined function

A NameError occurs when calling a function that is not defined or not imported.

** (Protocol.UndefinedError) protocol not implemented

A Protocol.UndefinedError occurs when calling a protocol function on a type that doesn't implement the protocol.

** (Task) error in async task

A Task error occurs when a spawned task crashes or returns an error during execution.
