A decimal precision error occurs when a numeric operation loses precision due to field size limitations.

Division by zero in COBOL occurs when a DIVIDE statement has a divisor of zero, causing an S0C8 ABEND.

Division error

A division error occurs when performing division by zero or an arithmetic operation that produces an invalid result.

Duplicate key error in COBOL

Duplicate key errors in COBOL occur when attempting to write a record with a key that already exists in an indexed file.

End of file

An end of file condition occurs when attempting to read past the last record in a file.

File not found in COBOL

File not found errors in COBOL occur when OPEN statement cannot locate the specified file.

File status error

A file status error occurs when a file operation fails, indicated by the FILE-STATUS field.

Index error in COBOL

Index errors in COBOL occur when using invalid subscript values or index registers that are out of range.

Invalid key error in COBOL

Invalid key errors in COBOL occur when referencing a record key that doesn't match the file's key structure.

Overflow error

An overflow error occurs when a numeric value exceeds the capacity of its receiving field.

Overflow in COBOL

Overflow errors in COBOL occur when arithmetic results exceed the capacity of the receiving field.

Record locked

A record locked error occurs when attempting to access a record that is locked by another process.

Runtime error in COBOL

Runtime errors in COBOL occur during program execution due to invalid operations, data exceptions, or resource limitations.

Subscript out of range

A subscript error occurs when accessing an array element with an index outside its declared bounds.

Syntax error

A syntax error occurs when the COBOL compiler encounters code that violates the language's syntax rules.

Undefined variable in COBOL

Undefined variable errors in COBOL occur when referencing a variable not declared in the Working-Storage or Linkage Section.
