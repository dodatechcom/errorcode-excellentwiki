Array bounds error

An array bounds error occurs when accessing an array element with an index outside its valid range.

EXC_BAD_ACCESS

An EXC_BAD_ACCESS occurs when accessing memory that has been freed or is invalid.

EXC_BREAKPOINT

An EXC_BREAKPOINT occurs when an assertion failure or breakpoint instruction is encountered.

key not found

A key not found error occurs when accessing a dictionary with a key that doesn't exist.

Memory warning

A memory warning indicates the system is running low on memory and may terminate the application.

message sent to nil

A message sent to nil occurs when calling a method on a nil object reference.

NSError domain error

An NSError occurs when an operation fails and provides detailed error information through domains and codes.

unrecognized selector sent to instance

An unrecognized selector error occurs when calling a method that the object doesn't implement.
