ClassCastException

A ClassCastException occurs when attempting to cast an object to a type it is not compatible with.

IllegalStateException

An IllegalStateException occurs when a method is invoked at an illegal or inappropriate time for the object's state.

IndexOutOfBoundsException

An IndexOutOfBoundsException occurs when accessing a collection with an index outside its valid range.

MatchError

A MatchError occurs when a match expression encounters a value that none of the cases handle.

NullPointerException

A NullPointerException occurs when attempting to access a method or field on a null reference.

OutOfMemoryException

An OutOfMemoryException occurs when the JVM cannot allocate enough memory.

StackOverflowException

A StackOverflowException occurs when the call stack exceeds its maximum depth due to infinite or very deep recursion.

UnsupportedOperationException

An UnsupportedOperationException occurs when an operation is not supported by the collection or object.
