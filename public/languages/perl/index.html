A bareword error occurs when using an unquoted identifier that Perl interprets as a bareword string under strict mode.

A reference error occurs when attempting to use a value as a reference when it's not a reference.

die error

A die error occurs when the die function is called, typically to indicate a fatal error condition.

Glob failed

A glob error occurs when pattern matching for filenames fails due to permission issues or invalid patterns.

syntax error at line X
