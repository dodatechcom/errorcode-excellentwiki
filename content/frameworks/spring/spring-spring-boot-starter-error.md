Starter not found.

## Common Causes

Wrong dependency.

## How to Fix

Add correct starter.

## Example

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```
