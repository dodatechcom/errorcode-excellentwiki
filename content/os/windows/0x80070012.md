# [Solution] Error 0x80070012 Windows 11/10 — Media is Write Protected Fix

Error 0x80070012 means the storage media you are trying to write to has write protection enabled. This prevents any data from being written, modified, or deleted on the affected drive or disk.

## Description

The full error message reads: **"The media is write protected."** This error commonly appears when trying to format a USB drive, SD card, or external hard disk, or when attempting to write data to a read-only optical disc. Write protection can be enabled both physically and through software settings.

## Common Causes

1. Physical write protection switch on USB drive or SD card is enabled
2. Disk attributes set to read-only through software
3. Group policy restrictions preventing writes to removable media
4. Registry settings enforcing write protection
5. Damaged or corrupted drive triggering read-only mode
6. Disk has reached its write limit (common on flash media)

## Solutions

### Solution 1: Check Physical Write Protection Switch

Many USB drives and SD cards have a physical switch:

1. Remove the USB drive or SD card from your computer
2. Look for a small **lock switch** on the side of the device
3. Slide the switch to the **unlocked** position
4. Reinsert the device and try writing again

### Solution 2: Remove Write Protection via Diskpart

Use the diskpart command-line tool to clear write protection:

1. Open **Command Prompt as Administrator**
2. Run these commands:
   ```
   diskpart
   list disk
   select disk X
   attributes disk clear readonly
   exit
   ```
3. Replace **X** with the number of your write-protected disk
4. Try writing to the disk again

### Solution 3: Check Disk Attributes in Disk Management

Verify and modify disk properties:

1. Press **Win + R**, type `diskmgmt.msc`, and press Enter
2. Right-click the affected disk and select **Properties**
3. Go to the **Policies** tab
4. Ensure **Removal policy** is set to **Quick removal** or **Better performance**
5. Click **OK** and try writing to the disk again
