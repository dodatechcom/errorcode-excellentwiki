# [Solution] Error 0x8007000E Windows 11/10 — Out of Memory Fix

Error 0x8007000E indicates that the system has run out of available memory. Unlike error 0x80070008, this error specifically means the memory allocation request failed because the system could not satisfy it from any available source.

## Description

The full error message reads: **"Out of memory."** This error commonly occurs during Windows Update, when running memory-intensive applications, or when too many processes compete for limited RAM. It can also appear when trying to open large files or during system-intensive operations.

## Common Causes

1. Insufficient physical RAM for the workload
2. Page file (virtual memory) set too small
3. Too many background applications consuming memory
4. A single application consuming excessive memory
5. 32-bit Windows with more than 4GB RAM not fully utilized
6. Memory management issues in installed software

## Solutions

### Solution 1: Increase the Page File Size

Expand virtual memory to supplement physical RAM:

1. Press **Win + R**, type `sysdm.cpl`, and press Enter
2. Go to **Advanced** > **Performance** > **Settings**
3. Click the **Advanced** tab and click **Change** under Virtual memory
4. Uncheck **Automatically manage paging file size for all drives**
5. Select your system drive and choose **Custom size**
6. Set Initial size to **1.5x your RAM** and Maximum size to **3x your RAM** in MB
7. Click **Set**, then **OK**, and restart your computer

### Solution 2: Add More Physical RAM

Upgrade your system's RAM if consistently running low:

1. Check your current RAM: **Task Manager** > **Performance** > **Memory**
2. Check maximum supported RAM for your motherboard
3. Purchase compatible RAM modules
4. Install the RAM and verify it is detected in Task Manager

### Solution 3: Close Background Programs and Startup Items

Reduce memory usage by limiting running processes:

1. Press **Ctrl + Shift + Esc** to open Task Manager
2. Sort processes by **Memory** usage
3. End unnecessary high-memory processes
4. Go to the **Startup** tab and disable non-essential startup items
5. Restart your computer
