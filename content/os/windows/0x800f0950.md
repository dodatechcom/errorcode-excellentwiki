# [Solution] Error 0x800F0950 Windows 11/10 — DISM Failed Fix

Error 0x800F0950 — DISM failed — occurs when the Deployment Imaging Service and Management Tool cannot complete its operation due to missing or inaccessible source files.

## Description

The full message reads:

> "DISM failed. No operation was performed (0x800F0950)"

This DISM error indicates that the tool could not locate or download the files needed to repair the Windows image. Common scenarios include:

- **Running DISM RestoreHealth** — The repair operation cannot find source files.
- **Windows feature installation** — Enabling features via DISM fails mid-operation.
- **Offline image servicing** — Servicing a mounted WIM image without proper sources.

## Common Causes

1. **No internet connection** — DISM cannot download files from Windows Update.
2. **Windows Update blocked** — Group Policy or firewall blocks access to Microsoft servers.
3. **Corrupted local source** — The installation media or local source path is damaged.
4. **Missing Windows Update files** — The required cab files are not available locally.
5. **Incorrect source path** — The `/Source` parameter points to an invalid location.

## Solutions

### Solution 1: Use Windows Installation Media as Source

Mount a Windows ISO and specify it as the source:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth /Source:D:\sources\install.wim /LimitAccess
```

Replace `D:` with your mounted ISO drive letter.

### Solution 2: Check Internet Connectivity

Ensure the machine has internet access and Windows Update is allowed through the firewall. If you are on a corporate network, try disconnecting from VPN.

### Solution 3: Use an Offline Installer

Download the required cab files from the Microsoft Update Catalog and place them in a local folder, then run:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth /Source:C:\temp\cab-files /LimitAccess
```

### Solution 4: Repair Using an Alternative Windows Version

If you have access to a different Windows machine with the same build, copy the `C:\Windows\WinSxS` folder and use it as the source:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth /Source:\\other-pc\c$\Windows\WinSxS /LimitAccess
```
