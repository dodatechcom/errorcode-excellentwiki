# [Solution] Error 0x80070643 Windows 11/10 — Windows Update Installation Failed Fix

Error 0x80070643 — Windows Update installation failed — occurs when a Windows Update cannot complete its installation. This is typically caused by corrupted update files or system component issues.

## Description

The full message reads:

> "Windows Update installation failed (0x80070643)"

This error appears when the Windows Update service cannot apply a downloaded update. Common scenarios include:

- **Windows Update installation** — A cumulative or feature update fails during installation.
- **Microsoft Store app updates** — Store updates fail with this error code.
- **Driver updates** — Optional driver updates through Windows Update fail to install.
- **Manual update installation** — Installing an MSU package manually returns this error.

## Common Causes

1. **Corrupted update files** — The downloaded update package is damaged.
2. **Corrupted system files** — Essential Windows files needed for the update are damaged.
3. **Insufficient disk space** — The system drive lacks space for the update installation.
4. **Windows Update components misconfigured** — Update-related services are in a bad state.
5. **Third-party software conflicts** — Antivirus or optimization tools interfere with updates.

## Solutions

### Solution 1: Run the Windows Update Troubleshooter

1. Press `Win + I` to open **Settings**.
2. Go to **System > Troubleshoot > Other troubleshooters**.
3. Run the **Windows Update** troubleshooter.
4. Apply any recommended fixes and restart.

### Solution 2: Run SFC and DISM

```cmd
sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth
sfc /scannow
```

Restart your computer after all scans complete.

### Solution 3: Reset Windows Update Components

```cmd
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 Catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver
```

### Solution 4: Free Up Disk Space

```cmd
cleanmgr /sageset:1
cleanmgr /sagerun:1
```

Ensure at least 20 GB of free space on the system drive.

### Solution 5: Manual Update via Microsoft Update Catalog

1. Note the KB number of the failing update from **Settings > Windows Update > Update history**.
2. Visit the **Microsoft Update Catalog** website.
3. Search for the KB number and download the correct version.
4. Run the downloaded `.msu` file manually.
