# [Solution] Error 0x800F0957 Windows 11/10 — DISM Source Download Failed Fix

Error 0x800F0957 — DISM source download failed — occurs when DISM cannot download source files from Windows Update, typically due to network restrictions or corrupted component store.

## Description

The full message reads:

> "DISM source download failed (0x800F0957)"

This error is closely related to 0x800F0956 and appears in similar DISM repair scenarios. Common scenarios include:

- **DISM RestoreHealth failures** — The repair tool cannot download needed files.
- **Windows Update connectivity issues** — Network restrictions prevent downloads.
- **Corporate WSUS environments** — WSUS does not serve the required files.
- **Offline repair attempts** — DISM cannot fall back to Windows Update.

## Common Causes

1. **Windows Update not accessible** — Network firewall or proxy blocks the connection.
2. **WSUS misconfiguration** — The WSUS server is not authorized for the required updates.
3. **Temporary Microsoft server issues** — Server-side outages prevent downloads.
4. **Missing required updates** — A prerequisite servicing stack update is missing.
5. **Network authentication required** — A captive portal or proxy authentication is blocking the request.

## Solutions

### Solution 1: Use a Local Source

Mount a Windows ISO matching your build and run:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth /Source:D:\sources\install.wim /LimitAccess
```

### Solution 2: Check Network and Proxy Settings

Verify internet access and configure proxy if needed:

```cmd
netsh winhttp show proxy
netsh winhttp set proxy proxy-server="http://your-proxy:port" bypass-list="*.local"
```

### Solution 3: Install Latest Servicing Stack Update

Download and install the latest SSU for your Windows version from the Microsoft Update Catalog, then retry DISM.

### Solution 4: Use an Alternative Windows Machine as Source

From a healthy Windows machine with the same build, share the WinSxS folder and use it:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth /Source:\\healthy-pc\WinSxS /LimitAccess
```
