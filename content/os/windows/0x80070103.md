# [Solution] Error 0x80070103 Windows 11/10 — Device Not Functioning Fix

Error 0x80070103 indicates that a hardware device attached to your system is not functioning properly. Windows has detected the device but cannot communicate with it or use it correctly due to driver or hardware issues.

## Description

The full error message reads: **"A device attached to the system is not functioning."** This error commonly appears in Device Manager with a yellow warning icon, during driver installations, or when connecting USB and peripheral devices. It suggests the device is recognized but cannot operate due to driver conflicts or hardware problems.

## Common Causes

1. Outdated or incompatible device drivers
2. Corrupted driver files
3. USB port or controller malfunction
4. Device hardware failure
5. Driver conflicts between multiple devices
6. Power management settings disabling the device

## Solutions

### Solution 1: Check Device Connection and Reconnect

Verify the physical connection:

1. Unplug the device from your computer
2. Wait 10 seconds
3. Plug the device into a **different USB port** (try ports on both front and back)
4. For internal devices, power off and reseat the component
5. Check if the device appears correctly in Device Manager

### Solution 2: Update or Reinstall Device Drivers

Fix driver-related issues:

1. Press **Win + X** and select **Device Manager**
2. Find the device with the yellow warning icon
3. Right-click it and select **Update driver**
4. Choose **Search automatically for updated drivers**
5. If no update is found, right-click and select **Uninstall device**
6. Restart your computer to reinstall the driver automatically
7. If the issue persists, download the latest driver from the manufacturer's website

### Solution 3: Run the Hardware Troubleshooter

Use Windows built-in troubleshooting:

1. Press **Win + I** to open Settings
2. Go to **System** > **Troubleshoot** > **Other troubleshooters**
3. Find **Hardware and Devices** and click **Run**
4. Follow the on-screen instructions
5. Apply any recommended fixes
6. Restart your computer
