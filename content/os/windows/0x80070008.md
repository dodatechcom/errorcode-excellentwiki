# [Solution] Error 0x80070008 Windows 11/10 — Not Enough Memory Fix

Error 0x80070008 indicates that Windows ran out of memory while trying to complete an operation. This typically occurs when the system's physical RAM and virtual memory are both insufficient to handle the demands of running applications and system processes.

## Description

The full error message reads: **"Not enough memory resources are available to process this command."** This error commonly appears when installing Windows updates, running large applications, or performing system operations that require significant memory allocation. It can affect both 32-bit and 64-bit versions of Windows 10 and 11.

## Common Causes

1. Insufficient physical RAM installed on the system
2. Page file (virtual memory) configured too small
3. Too many resource-heavy programs running simultaneously
4. Memory leak in a running application consuming available RAM
5. System reserved memory being used by hardware or drivers
6. Malware consuming memory resources

## Solutions

### Solution 1: Increase Virtual Memory (Page File)

Open the System Properties and increase the page file size:

1. Press **Win + R**, type `sysdm.cpl`, and press Enter
2. Go to the **Advanced** tab and click **Settings** under Performance
3. Click **Advanced** tab, then **Change** under Virtual memory
4. Uncheck **Automatically manage paging file size**
5. Select your system drive, choose **Custom size**
6. Set Initial size to **4096 MB** and Maximum size to **8192 MB** (or 1.5x your RAM)
7. Click **Set**, then **OK**, and restart your computer

### Solution 2: Close Unnecessary Programs

Free up memory by closing programs you are not using:

1. Press **Ctrl + Shift + Esc** to open Task Manager
2. Click **More details** if needed
3. Go to the **Processes** tab and sort by **Memory**
4. Right-click resource-heavy applications and select **End task**
5. Disable unnecessary startup programs under the **Startup** tab

### Solution 3: Run Windows Memory Diagnostic

Check for physical memory errors:

1. Press **Win + R**, type `mdsched.exe`, and press Enter
2. Select **Restart now and check for problems**
3. Your computer will restart and run the memory test
4. Review the results after the test completes

### Solution 4: Check for Memory Leaks

Identify applications leaking memory:

1. Open **Task Manager** and go to the **Processes** tab
2. Sort by **Memory** usage
3. Monitor processes over time; if a process continuously increases memory usage without releasing it, it may have a memory leak
4. Update or reinstall the offending application
