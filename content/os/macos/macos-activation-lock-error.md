# macOS Activation Lock Error -- Mac Is Locked to Previous Owner's Apple ID

Activation Lock prevents unauthorized use of a Mac by requiring the previous owner's Apple ID and password. When buying a used Mac, activation lock can prevent setup.

## Common Causes
- Previous owner did not sign out of iCloud before selling
- Find My Mac was not disabled
- Mac was not properly erased before being sold
- Stolen Mac with activation lock enabled
- Previous owner is unreachable

## How to Fix
1. Contact the previous owner and ask them to remove activation lock
2. Previous owner can remove it via iCloud.com > Find My > All Devices
3. Bring the Mac and proof of purchase to an Apple Store
4. Check Apple's activation lock status before purchasing a used Mac
5. If the Mac was stolen, contact law enforcement

```bash
# Check activation lock status
# The Mac will show an Activation Lock screen during setup
# You cannot bypass this without the Apple ID credentials
```

## Examples

```bash
# Previous owner can remove activation lock remotely:
# 1. Sign in to iCloud.com
# 2. Go to Find My > All Devices
# 3. Select the Mac and click "Remove from Account"
```

This error is common when purchasing a used Mac that was not properly erased, when the previous owner did not sign out of iCloud, or when a Mac was stolen.
