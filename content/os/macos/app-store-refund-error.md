# App Store Refund Error — Cannot Request Refund

Fix macOS App Store refund issue: cannot request refund, refund not appearing, App Store refund denied or delayed.

## Common Causes

- Refund request submitted outside allowed time window
- App Store refund policy does not cover the purchase type
- Refund request denied by Apple due to usage evidence
- Payment method cannot receive refunds

## How to Fix

### 1. Request Refund via Report a Problem

```bash
open https://reportaproblem.apple.com
# Sign in with Apple ID → Find purchase → Request Refund
```

### 2. Check Refund Status

```bash
# reportaproblem.apple.com → Check status of previous refund requests
```

### 3. Contact Apple Support

```bash
# Visit https://support.apple.com/contact → Request refund assistance
```

### 4. Check Payment Method for Refund

```bash
# Check card statement for refund credit after approval
```

## Common Scenarios

This error commonly occurs when:

- Cannot find purchase in Report a Problem to request refund
- Refund request submitted but no response after several days
- Refund denied with 'not eligible for refund' message
- Refund approved but credit not appearing on payment method

## Prevent It

- Submit refund requests within 90 days of purchase
- Use reportaproblem.apple.com for fastest refund processing
- Keep receipts and order confirmations for refund requests
- Contact Apple Support directly if online refund is denied
