# VBA Syntax Error

VBA parser encounters invalid syntax.

### Common Causes
Missing keyword; wrong operator; missing parentheses

### How to Fix
```vba
Dim x As Integer
x = 10
```

### Examples
```vba
Dim result As Double
result = CDbl(TextBox1.Value)
```
