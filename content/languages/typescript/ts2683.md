# TS2683 — 'this' implicitly has type 'any'

TS2683 occurs when `noImplicitThis` is enabled (part of `strict` mode) and TypeScript can't determine the type of `this` in a function. This commonly happens with standalone functions, callbacks, and untyped method references.

## Common Causes

```typescript
// Cause 1: Standalone function without 'this' parameter
function greet() {
  console.log(this.name); // TS2683: 'this' implicitly has type 'any'
}

// Cause 2: Callback losing 'this' context
class Timer {
  seconds = 0;
  start() {
    setInterval(function () {
      this.seconds++; // TS2683: 'this' is 'any' inside callback
    }, 1000);
  }
}

// Cause 3: Passing method as callback without binding
class Logger {
  log() {
    console.log("logged");
  }
}
const fn = new Logger().log;
fn(); // TS2683: 'this' is 'any'

// Cause 4: Object literal with untyped 'this'
const obj = {
  value: 10,
  getValue() {
    return this.value; // TS2683 if noImplicitThis is on
  }
};
```

## Solutions

### Fix 1: Add a 'this' parameter to the function

```typescript
// Wrong
function greet() {
  console.log(this.name);
}

// Correct
interface ThisContext {
  name: string;
}
function greet(this: ThisContext) {
  console.log(this.name);
}
```

### Fix 2: Use arrow functions to preserve 'this'

```typescript
// Wrong
class Timer {
  seconds = 0;
  start() {
    setInterval(function () {
      this.seconds++;
    }, 1000);
  }
}

// Correct — arrow function captures 'this' lexically
class Timer {
  seconds = 0;
  start() {
    setInterval(() => {
      this.seconds++;
    }, 1000);
  }
}
```

### Fix 3: Use .bind() when passing methods as callbacks

```typescript
class Logger {
  log() {
    console.log("logged");
  }
}
const logger = new Logger();
const fn = logger.log.bind(logger);
fn();
```

### Fix 4: Type 'this' in arrow function properties

```typescript
const obj = {
  value: 10,
  getValue: function (this: { value: number }) {
    return this.value;
  }
};
```

## Examples

```typescript
// Example 1: jQuery-style callback
$(".button").on("click", function () {
  $(this).addClass("active"); // TS2683: 'this' is HTMLElement, not any
  // Fix: add 'this: HTMLElement' parameter
});

// Example 2: Event handler
document.addEventListener("click", function (e) {
  console.log(this.id); // TS2683: 'this' is Document, not HTMLElement
});
```
