# TS2728 — A module cannot be aliased to a non-module type

TS2728 occurs when an `import ... = require(...)` or `import ... = ...` statement tries to alias a module to a type that is not a module. The right-hand side must resolve to a module (a file with exports).

## Common Causes

```typescript
// Cause 1: Importing a non-module file with import =
import MyLib = require("./not-a-module"); // TS2728 if file has no exports

// Cause 2: Importing a plain variable file
// types.ts: export const MY_TYPE = "string";
import Type = require("./types"); // TS2728 — this works if it's a module, but if the file is ambient without module exports:

// Cause 3: Importing a file that only has declarations
// ambient.d.ts: declare const helper: () => void;
import helper = require("./ambient"); // TS2728 — not a module

// Cause 4: Aliasing a namespace import to a type
import * as lib from "./mylib";
import Alias = lib.NonExistent; // TS2728

// Cause 5: Importing a script file (no top-level exports)
// script.ts: console.log("hello"); (no export)
import Script = require("./script"); // TS2728
```

## Solutions

### Solution 1: Ensure the file is a module (has exports)

```typescript
// Wrong — file has no exports
// not-a-module.ts
const x = 42;

// Correct — add an export
// not-a-module.ts
export const x = 42;
```

### Solution 2: Use regular import instead of import =

```typescript
// Wrong
import MyLib = require("./mylib"); // TS2728

// Correct
import MyLib from "./mylib"; // ES module default import
// or
import * as MyLib from "./mylib";
```

### Solution 3: Add a proper export to the file

```typescript
// Wrong — ambient file without module declaration
// ambient.d.ts
declare const helper: () => void;

// Correct — make it a module
// ambient.d.ts
declare module "ambient" {
  export function helper(): void;
}
```

## Examples

```typescript
// Valid module (has exports)
// math.ts
export function add(a: number, b: number): number {
  return a + b;
}

// Valid import =
import math = require("./math");
console.log(math.add(1, 2));

// Valid — re-exporting
export import math2 = require("./math");
```
