The error "[Solution] TypeScript TS1005 — ';' expected" occurs when ts1005 occurs when a semicolon is expected after a statement.

## Solution

Add the missing semicolon.

## Code Example

```typescript
const x = 10
const y = 20 // TS1005 - missing semicolon after previous statement
```
