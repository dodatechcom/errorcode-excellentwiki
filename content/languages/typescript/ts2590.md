# TS2590 — Expression produces a union type that is too complex

TS2590 occurs when TypeScript computes a union type that has too many members, typically from conditional types, distributive conditional types, or recursive mapped types that generate large numbers of union variants.

## Common Causes

```typescript
// Cause 1: Distributive conditional type over large union
type LargeUnion = 1 | 2 | 3 | ... | 100; // large union
type Process<T> = T extends number ? { value: T } : never;
type Result = Process<LargeUnion>; // TS2590 — too many union members

// Cause 2: Recursive template literal with many permutations
type Color = "red" | "green" | "blue";
type Variant = "light" | "dark" | "bright";
type Shades = `${Variant}-${Color}`; // OK — only 6 members
// But with 20+ variants it can blow up

// Cause 3: Deeply nested conditional types producing unions
type DeepUnion<T> = T extends { a: infer U }
  ? U extends string ? "string" : "other"
  : never;
// Applied to a large recursive type
```

## Solutions

### Fix 1: Reduce the size of the input union

```typescript
// Wrong
type AllKeys = keyof SomeHugeType;
type Processed = Process<AllKeys>; // TS2590

// Correct — process subsets
type KeySubset = "id" | "name" | "email";
type Processed = Process<KeySubset>;
```

### Fix 2: Use non-distributive conditional types

```typescript
// Distributive (default):
type Distribute<T> = T extends any ? SomeMapping<T> : never;

// Non-distributive — wrap in tuple:
type NonDistribute<T> = [T] extends [any] ? SomeMapping<T> : never;
```

### Fix 3: Simplify the type logic

```typescript
// Instead of generating a union of all possible combinations,
// use a mapped type or interface:

// Wrong — generates huge union
type ButtonStyles = {
  [K in Color]: Record<K, string>;
}[Color];

// Better — use a concrete mapped type
type ButtonStyles = Record<Color, string>;
```

## Examples

```typescript
// Example: Combining multiple enums into a union type
enum A { X, Y, Z }
enum B { P, Q, R }
type Combined = `${A}_${B}`; // "X_P" | "X_Q" | "X_R" | "Y_P" | ... -- 9 members OK

// But if A and B each had 20 members, Combined would have 400 members
// and could trigger TS2590.

// Fix: Use a function with overloads instead of a union type
```
