# TS2730 — An interface can only extend an object type or intersection of object types

TS2730 occurs when an interface tries to extend a type that is not an object type — such as a primitive type, union type, or a type alias that resolves to a non-object. Interfaces can only extend other interfaces, object types, or intersection types of object types.

## Common Causes

```typescript
// Cause 1: Extending a primitive type alias
type MyString = string;
interface BetterString extends MyString {} // TS2730

// Cause 2: Extending a union type
type Animal = { name: string } | { species: string };
interface Dog extends Animal {} // TS2730

// Cause 3: Extending a type alias that is a primitive
type ID = number;
interface TypedID extends ID {} // TS2730

// Cause 4: Extending a type that resolves to a union
type Result = Success | Failure;
interface Success { ok: true; }
interface Failure { ok: false; }
interface DetailedResult extends Result {} // TS2730

// Cause 5: Extending a type alias that is an array
type StringArray = string[];
interface CustomArray extends StringArray {} // TS2730
```

## Solutions

### Solution 1: Extend an interface or object type instead

```typescript
// Wrong
type MyString = string;
interface BetterString extends MyString {} // TS2730

// Correct
interface BetterString {
  value: string;
  toUpperCase(): string;
}
```

### Solution 2: Use intersection types for unions

```typescript
// Wrong
type Animal = { name: string } | { species: string };
interface Dog extends Animal {} // TS2730

// Correct — extend the common interface
interface HasName {
  name: string;
}
interface Dog extends HasName {
  breed: string;
}
```

### Solution 3: Use type alias intersection instead of interface extends

```typescript
// Wrong
type ID = number;
interface TypedID extends ID {} // TS2730

// Correct
type TypedID = number & { __brand: never };
// or
interface TypedID {
  value: number;
  __brand: "typed-id";
}
```

### Solution 4: Extend from a concrete interface

```typescript
// Wrong
type Result = Success | Failure;
interface DetailedResult extends Result {} // TS2730

// Correct
interface BaseResult {
  ok: boolean;
}
interface DetailedResult extends BaseResult {
  details: string;
}
```

## Examples

```typescript
// Valid — extending an interface
interface Printable {
  print(): void;
}

interface Loggable {
  log(): void;
}

// Valid — extending another interface
interface Document extends Printable, Loggable {
  title: string;
  content: string;
}

// Valid — extending an object type literal
interface Config {
  host: string;
  port: number;
}

interface ExtendedConfig extends Config {
  debug: boolean;
}
```
