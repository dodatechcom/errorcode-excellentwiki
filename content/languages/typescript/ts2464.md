# TS2464 — A class can only implement an identifier/qualified-name

TS2464 occurs when you use `implements` with an expression that isn't a valid identifier or qualified name. You can only implement named interfaces or classes, not object literals, unions, or computed values.

## Common Causes

```typescript
// Cause 1: Implementing an object literal
class MyClass implements { name: string; age: number } { // TS2464
  name = "";
  age = 0;
}

// Cause 2: Implementing a union type
interface A { a: string; }
interface B { b: string; }
class MyClass implements A | B { // TS2464
  a = "";
}

// Cause 3: Implementing an expression
const myInterface = { x: number };
class MyClass implements myInterface {} // TS2464

// Cause 4: Implementing an inline type without a name
class MyClass implements { id: number } {} // TS2464
```

## Solutions

### Fix 1: Define an interface and implement it

```typescript
// Wrong
class MyClass implements { name: string } {}

// Correct
interface Named {
  name: string;
}
class MyClass implements Named {
  name = "";
}
```

### Fix 2: Only implement one interface (not a union)

```typescript
// Wrong
class MyClass implements A | B {}

// Correct — implement both separately
class MyClass implements A, B {
  a = "";
  b = "";
}
```

### Fix 3: Use interface reference instead of inline type

```typescript
// Wrong
const myInterface = { x: number };
class MyClass implements myInterface {}

// Correct
interface HasX {
  x: number;
}
class MyClass implements HasX {
  x = 0;
}
```

### Fix 4: Implement named type aliases

```typescript
type Printable = { print(): void };
class MyClass implements Printable {
  print() { console.log("printed"); }
}
```

## Examples

```typescript
// Example 1: Trying to implement a generic type without instantiation
interface Container<T> { value: T; }
class MyClass implements Container { // TS2464 — missing type argument
  value = "";
}

// Fix:
class MyClass implements Container<string> {
  value = "";
}

// Example 2: Implementing a mapped type
type Options = { [key: string]: string };
class Config implements Options {} // TS2464 — can't implement mapped type directly
```
