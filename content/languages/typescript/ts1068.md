# TS1068 — Cannot find name

TS1068 was the earlier error code for what is now TS2304 (Cannot find name) in older versions of TypeScript. It indicates the compiler cannot resolve a name reference to any declaration in scope.

## Common Causes

```typescript
// Cause 1: Reference to undefined variable
console.log(undefinedVariable); // TS1068

// Cause 2: Typo in variable or function name
function greet() { return "hello"; }
console.log(grett()); // TS1068

// Cause 3: Using a name before declaration (if not hoisted)
console.log(x); // TS1068
let x = 5;

// Cause 4: Missing import
// utils.ts exports helper, but not imported
helper(); // TS1068
```

## Solutions

### Fix 1: Declare the variable or function

```typescript
// Wrong
console.log(myVar);

// Correct
const myVar = 42;
console.log(myVar);
```

### Fix 2: Fix typos

```typescript
// Wrong
const user = { name: "Alice" };
console.log(usre.name);

// Correct
console.log(user.name);
```

### Fix 3: Add the missing import

```typescript
// Wrong
helper();

// Correct
import { helper } from "./utils";
helper();
```

## Examples

```typescript
// Example 1: Global reference without declaration
// In a browser environment:
alert("hello"); // OK — declared by lib.dom.d.ts
myGlobalFunc(); // TS1068 — not declared

// Example 2: Reference to deleted variable
let temp = 10;
// ... later
console.log(tmp); // TS1068 — typo
```
