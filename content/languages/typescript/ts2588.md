# TS2588 — Cannot assign to 'const' variable

TS2588 occurs when you try to reassign a variable declared with `const`. Unlike `let`, `const` variables cannot be reassigned after initialization. This is a fundamental JavaScript/TypeScript rule enforced at compile time.

## Common Causes

```typescript
// Cause 1: Reassigning a const variable
const x = 10;
x = 20; // TS2588: Cannot assign to 'x' because it is a constant

// Cause 2: Trying to reassign in a loop
const counter = 0;
for (let i = 0; i < 5; i++) {
  counter++; // TS2588
}

// Cause 3: Reassigning a function parameter (const in destructuring)
function process({ id, name }: { id: number; name: string }) {
  id = 0; // TS2588: 'id' is a const
}

// Cause 4: Increment/decrement operator on const
const count = 0;
count++; // TS2588
```

## Solutions

### Fix 1: Use 'let' for variables that need reassignment

```typescript
// Wrong
const x = 10;
x = 20; // TS2588

// Correct
let x = 10;
x = 20; // OK
```

### Fix 2: Use 'let' for counters

```typescript
// Wrong
const counter = 0;
for (let i = 0; i < 5; i++) {
  counter++; // TS2588
}

// Correct
let counter = 0;
for (let i = 0; i < 5; i++) {
  counter++;
}
```

### Fix 3: Use a new variable instead of reassignment

```typescript
// Wrong
const count = 0;
count++;

// Correct — create a new value
const count = 0;
const newCount = count + 1;
```

### Fix 4: Use mutable objects with const

```typescript
// Const prevents reassignment, not mutation
const arr = [1, 2, 3];
arr.push(4); // OK — mutation is allowed
arr = [5, 6, 7]; // TS2588 — reassignment is not

// If you need to "replace" the array
const arr2 = [...arr, 4]; // New array, same reference
```

## Examples

```typescript
// Example 1: React state with const
function Counter() {
  const [count, setCount] = useState(0);
  // count++; // TS2588 — can't reassign const
  setCount(count + 1); // Correct — use setter
}

// Example 2: Trying to swap values
let a = 1;
let b = 2;
const temp = a;
a = b;
b = temp; // OK — temp is const but we're not reassigning it
// But: temp = 5; // TS2588

// Example 3: Destructured const
const [first, second] = [1, 2];
// first = 10; // TS2588
```
