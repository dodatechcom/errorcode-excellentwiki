# TS2310 — Type 'X' recursively references itself

TS2310 occurs when a type directly or indirectly refers to itself in a way that creates an infinite recursion. TypeScript cannot compute the type because it never terminates. This is different from valid recursive types like `type Tree = { value: string; children: Tree[] }`.

## Common Causes

```typescript
// Cause 1: Direct self-reference in a type alias without termination
type Bad = Bad[]; // TS2310 — no base case

// Cause 2: Circular reference between two types
type A = B;
type B = A; // TS2310 — circular

// Cause 3: Self-referencing interface
interface Node {
  value: number;
  child: Node; // This is fine, but if the reference is not structurally terminating
}
// A pathological case:
type Infinite = { next: Infinite } & { extra: Infinite };

// Cause 4: Generic type that references itself without a base case
type Recursive<T> = Recursive<T>; // TS2310

// Cause 5: Mapped type that maps to itself
type SelfRef<T> = {
  [K in keyof T]: SelfRef<T[K]>; // TS2310 — no termination
};
```

## Solutions

### Solution 1: Add a base case or stopping condition

```typescript
// Wrong
type Bad = Bad[]; // TS2310 — no base case

// Correct
type Good = string | Good[]; // string is the base case
```

### Solution 2: Break the circular reference

```typescript
// Wrong
type A = B;
type B = A; // TS2310

// Correct
type A = string;
type B = number;
```

### Solution 3: Use an interface for self-referencing structures

```typescript
// Wrong
type Tree = { value: string; children: Tree }; // TS2310 if not array

// Correct — interface allows structural recursion
interface TreeNode {
  value: string;
  children: TreeNode[];
}

// Or using a type alias with an array
type Tree = { value: string; children: Tree[] };
```

### Solution 4: Use conditional types with a base case

```typescript
// Wrong
type Nested<T> = Nested<T>; // TS2310

// Correct
type Flatten<T> = T extends Array<infer U> ? Flatten<U> : T; // Terminates when T is not an array
```

## Examples

```typescript
// Valid recursive type — JSON value
type JSONValue =
  | string
  | number
  | boolean
  | null
  | JSONValue[]
  | { [key: string]: JSONValue };

// Valid recursive type — tree
type Tree<T> = {
  value: T;
  children: Tree<T>[];
};

// Valid recursive type — deeply nested object
type NestedObject = {
  [key: string]: string | NestedObject;
};

const data: NestedObject = {
  a: "hello",
  b: {
    c: "world",
    d: {
      e: "deep",
    },
  },
};
```
