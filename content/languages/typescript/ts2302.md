# TS2302 — Static members cannot reference class type parameters

Type class parameters (generics) are per-instance. Static members belong to the class constructor itself, not to any instance, so they cannot reference the class's type parameters.

## Common Causes

```typescript
// Cause 1: Static method using class type parameter
class Container<T> {
  static create(): T { // TS2302
    return {} as T;
  }
}

// Cause 2: Static property typed with class type parameter
class Factory<T> {
  static default: T; // TS2302
}

// Cause 3: Static accessor using class type parameter
class Store<T> {
  static get item(): T { // TS2302
    return {} as T;
  }
}

// Cause 4: Static method parameter typed with class type parameter
class Repository<T> {
  static add(item: T): void {} // TS2302
}
```

## Solutions

### Solution 1: Make the static method generic independently

```typescript
// Wrong
class Container<T> {
  static create(): T { // TS2302
    return {} as T;
  }
}

// Correct
class Container<T> {
  create(): T {
    return {} as T;
  }
}

// If you need a static-like factory, add its own generic
class Container {
  static create<U>(): U {
    return {} as U;
  }
}
```

### Solution 2: Use the static method's own type parameter

```typescript
// Wrong
class Factory<T> {
  static default: T; // TS2302
}

// Correct — use a separate generic for static context
class Factory<T> {
  static createDefault<U>(): U {
    return {} as U;
  }
}
```

### Solution 3: Move the typed member to an instance

```typescript
// Wrong
class Store<T> {
  static get item(): T { // TS2302
    return {} as T;
  }
}

// Correct
class Store<T> {
  private _item!: T;
  get item(): T {
    return this._item;
  }
}
```

## Examples

```typescript
class Registry<T> {
  private items: T[] = [];

  // Instance method — can use T
  add(item: T): void {
    this.items.push(item);
  }

  // Static method — cannot use T, uses own generic
  static fromArray<U>(arr: U[]): Registry<U> {
    const reg = new Registry<U>();
    arr.forEach(item => reg.add(item));
    return reg;
  }
}

const nums = Registry.fromArray([1, 2, 3]); // Registry<number>
```
