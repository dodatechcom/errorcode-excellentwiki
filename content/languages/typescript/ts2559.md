# TS2559 — Type has no properties in common with type

TS2559 occurs when TypeScript checks type compatibility between two types and finds no overlapping properties. This typically happens during excess property checks or when using `&` intersection types with completely incompatible types.

## Common Causes

```typescript
// Cause 1: Intersection of incompatible object types
type A = { a: number };
type B = { b: string };
type C = A & B; // OK — results in { a: number; b: string }

// But if types have conflicting properties:
type A = { a: number; common: string };
type B = { b: string; common: number };
type C = A & B; // TS2559 — 'common' property types conflict

// Cause 2: Assigning unrelated types
interface Dog { bark(): void; }
interface Cat { meow(): void; }
const pet: Dog = { meow() {} } as Cat; // TS2559

// Cause 3: Object literal with completely unrelated properties
interface Config { url: string; }
const cfg: Config = { port: 8080 }; // TS2559 — no 'url' property
```

## Solutions

### Fix 1: Fix property names to match

```typescript
// Wrong
interface Config { url: string; }
const cfg: Config = { port: 8080 };

// Correct
const cfg: Config = { url: "http://localhost" };
```

### Fix 2: Use a union type if appropriate

```typescript
interface Dog { bark(): void; }
interface Cat { meow(): void; }
type Pet = Dog | Cat;

function handlePet(pet: Pet) {
  if ("bark" in pet) {
    pet.bark();
  }
}

const myPet: Pet = { meow() {} }; // OK — matches Cat part of union
```

### Fix 3: Ensure intersection types have compatible properties

```typescript
// Wrong
type A = { common: number };
type B = { common: string };
type C = A & B; // TS2559

// Correct — make properties compatible
type A = { common: number | string };
type B = { common: number | string };
type C = A & B; // OK
```

## Examples

```typescript
// Example 1: Completely different shapes
interface Square { size: number; }
interface Circle { radius: number; }
type Shape = Square | Circle;

const shape: Shape = { size: 10 }; // OK
const shape2: Shape = { width: 5 }; // TS2559 — no common properties
```
