# TS2315 — Type 'X' is not generic

TS2315 occurs when you try to use a type with type arguments, but the type is not defined as a generic type. Only types with type parameters can accept type arguments.

## Common Causes

```typescript
// Cause 1: Using type arguments on a non-generic type
type Name = string;
const x: Name<string> = "hello"; // TS2315 — Name is not generic

// Cause 2: Using type arguments on a built-in type
const arr: Array<string, number> = [1, 2]; // TS2315 — Array only takes 1 arg

// Cause 3: Using type arguments on an interface that has no parameters
interface Config {
  host: string;
  port: number;
}

const c: Config<string> = { host: "a", port: 1 }; // TS2315

// Cause 4: Using type arguments on a class that has no parameters
class Person {
  name!: string;
}

const p: Person<string> = new Person(); // TS2315

// Cause 5: Incorrectly parameterizing a type alias
type StringOrNumber = string | number;
const val: StringOrNumber<boolean> = true; // TS2315
```

## Solutions

### Solution 1: Remove the type arguments

```typescript
// Wrong
type Name = string;
const x: Name<string> = "hello"; // TS2315

// Correct
const x: Name = "hello";
```

### Solution 2: Make the type generic if needed

```typescript
// Wrong
interface Config {
  host: string;
  port: number;
}
const c: Config<string> = { host: "a", port: 1 }; // TS2315

// Correct — add a type parameter
interface Config<T = string> {
  host: T;
  port: number;
}
const c: Config<number> = { host: 127, port: 1 };
```

### Solution 3: Use the correct number of type arguments

```typescript
// Wrong
const arr: Array<string, number> = [1, 2]; // TS2315

// Correct
const arr: Array<string> = ["hello", "world"];
```

## Examples

```typescript
// Non-generic types — no type arguments
const x: string = "hello";
const y: number = 42;
const z: boolean = true;

// Generic types — must use type arguments
interface Response<T> {
  data: T;
  status: number;
}

const r: Response<string> = { data: "ok", status: 200 };

// Built-in generic types
const promise: Promise<number> = Promise.resolve(42);
const map: Map<string, number> = new Map([["a", 1]]);
```
