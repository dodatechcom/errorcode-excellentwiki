# TS2542 — Cannot read property 'X' of type 'Y'

TS2542 occurs when you try to read a property from a type that doesn't allow property access, typically with `Readonly<T>` types or when accessing properties on types that lack index signatures. This is less common than TS2540 and usually involves specific readonly constraints.

## Common Causes

```typescript
// Cause 1: Accessing property on wrong type
interface ReadonlyUser {
  readonly name: string;
  readonly age: number;
}
const user: ReadonlyUser = { name: "Alice", age: 30 };
const data: ReadonlyUser = user;
// TS2542 if you try to access data on a type that doesn't expose it

// Cause 2: Indexed access on a readonly tuple
const tuple: readonly [string, number] = ["hello", 42];
const val: string = tuple[0]; // OK
tuple[0] = "world"; // TS2540 (write), not TS2542

// Cause 3: Using 'in' operator on type without property
type Empty = {};
const x: Empty = {};
"prop" in x; // No error, but...

// Cause 4: Readonly mapped type access
type ReadonlyObj = Readonly<{ x: number; y: number }>;
const obj: ReadonlyObj = { x: 1, y: 2 };
```

## Solutions

### Fix 1: Use the correct type that allows property access

```typescript
// If you need to read properties, ensure the type exposes them
interface User {
  name: string;
  age: number;
}
const user: User = { name: "Alice", age: 30 };
console.log(user.name); // OK
```

### Fix 2: Use a mutable type when mutation is needed

```typescript
// Wrong — readonly prevents writes
type ReadonlyUser = Readonly<{ name: string }>;

// Correct — use mutable type for mutable operations
type User = { name: string };
```

### Fix 3: Cast to a type with the needed property

```typescript
interface Base {
  id: number;
}
interface Extended extends Base {
  name: string;
}
const obj: Base = { id: 1 };
(obj as Extended).name; // Access via assertion
```

### Fix 4: Use type narrowing for conditional access

```typescript
function getValue(obj: unknown) {
  if (typeof obj === "object" && obj !== null && "name" in obj) {
    return (obj as { name: string }).name; // Safe access
  }
}
```

## Examples

```typescript
// Example 1: Readonly array methods
const arr: readonly number[] = [1, 2, 3];
arr.filter(x => x > 1); // OK — read-only methods allowed
arr.sort(); // TS2542 — mutating method not allowed on readonly array

// Example 2: Frozen object
const frozen = Object.freeze({ config: { debug: true } });
frozen.config = { debug: false }; // TS2542 / TS2540
```
