# CSS Word Spacing Error

Word-spacing not working as expected.

### Common Causes
Values too large; wrong context

### How to Fix
```css
word-spacing: 0.5em;
```

### Examples
```css
.spaced-words {
  word-spacing: 2px;
}
```
