# CSS Element Not Found

CSS element selector does not match.

### Common Causes
Wrong tag name; element not in DOM

### How to Fix
```css
/* Ensure element exists */
article { max-width: 800px; }
```

### Examples
```css
section { padding: 2rem; }
```
