# CSS Class Not Found

CSS class selector does not match any element.

### Common Causes
Class name misspelled; not applied to element

### How to Fix
```css
/* Check HTML for correct class */
.my-class { color: red; }
```

### Examples
```css
.highlight { background: yellow; }
```
