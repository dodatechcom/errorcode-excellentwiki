# Provisioning Profile Team ID Mismatch

Team ID must match between the provisioning profile and your project's signing configuration. Mismatches prevent successful code signing.

## Common Causes
- Wrong team selected in Xcode signing settings
- Profile generated by a different team
- Enterprise vs personal account confusion
- Multiple teams in the Apple Developer portal

## How to Fix
1. Verify your Team ID in the developer portal
2. Select the correct team in Target > Signing & Capabilities
3. Regenerate the profile with the correct team
4. Check DEVELOPMENT_TEAM in build settings

```swift
// Find your Team ID:
// Xcode > Settings > Accounts > Select team > View Details
// Team ID is displayed (e.g., ABCDE12345)

// Or check in Build Settings:
// DEVELOPMENT_TEAM = ABCDE12345
```

## Examples
```swift
// Example: Setting team via xcconfig file
// Create: Team.xcconfig
// DEVELOPMENT_TEAM = ABCDE12345

// Include in project:
// Build Settings > Based on configuration file
// Select Team.xcconfig for Debug and Release

// Verify with xcodebuild:
// $ xcodebuild -showBuildSettings | grep DEVELOPMENT_TEAM
```
