# E0626: Trait Object Not Object-Safe

A trait cannot be used as a trait object — it has non-object-safe methods.

## Common Causes

```rust
trait MyTrait {
    fn method(&self) -> Self where Self: Sized; // ERROR: not object-safe
}
let obj: Box<dyn MyTrait> = Box::new(my_impl); // ERROR
```

## How to Fix

1. **Remove Self: Sized bounds**

```rust
trait MyTrait {
    fn method(&self);
}
```

2. **Use generics instead of trait objects**

```rust
fn do_thing<T: MyTrait>(item: &T) {
    item.method();
}
```

3. **Separate object-safe and non-object-safe parts**

```rust
trait MyTrait {
    fn method(&self);
}
trait MyTraitExt: MyTrait {
    fn generic_method(&self) where Self: Sized;
}
```

## Examples

```rust
trait Drawable {
    fn draw(&self);
}

struct Circle { radius: f64 }
struct Square { side: f64 }

impl Drawable for Circle {
    fn draw(&self) { println!("Drawing circle r={}", self.radius); }
}
impl Drawable for Square {
    fn draw(&self) { println!("Drawing square s={}", self.side); }
}

fn main() {
    let shapes: Vec<Box<dyn Drawable>> = vec![
        Box::new(Circle { radius: 1.0 }),
        Box::new(Square { side: 2.0 }),
    ];
    for s in &shapes { s.draw(); }
}
```
