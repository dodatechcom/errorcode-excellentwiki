# E0502: Cannot Borrow as Mutable

A value is already borrowed immutably — mutable access requires exclusive ownership.

## Common Causes

```rust
let mut v = vec![1, 2, 3];
let first = &v[0]; // immutable borrow
v.push(4); // ERROR: cannot mutably borrow while immutable borrow exists
```

## How to Fix

1. **Reorder operations**

```rust
let mut v = vec![1, 2, 3];
v.push(4); // mutate first
let first = &v[0]; // then borrow
```

2. **Copy the value**

```rust
let mut v = vec![1, 2, 3];
let first = v[0]; // Copy the value
v.push(4);
println!("{}", first);
```

3. **Use split_at_mut**

```rust
let mut v = vec![1, 2, 3, 4, 5];
let (left, right) = v.split_at_mut(3);
// left and right can be mutated independently
```

## Examples

```rust
fn main() {
    let mut numbers = vec![1, 2, 3, 4, 5];
    let len = numbers.len();
    numbers.push(6);
    println!("Added element, now {} elements: {:?}", len + 1, numbers);
}
```
