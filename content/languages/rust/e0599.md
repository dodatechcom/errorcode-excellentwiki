# E0599: No Method Found

The type does not have the requested method — check imports and trait implementations.

## Common Causes

```rust
let s = "hello";
s.to_uppercase(); // Works
s.push_str("!"); // ERROR: &str has no push_str
```

## How to Fix

1. **Use the correct type**

```rust
let mut s = String::from("hello");
s.push_str("!"); // String has push_str
```

2. **Import the trait providing the method**

```rust
use std::io::Write; // must import to get .write() on TcpStream
```

3. **Check method availability**

```rust
// For &str, use String methods after conversion
let s: String = "hello".to_string();
```

## Examples

```rust
fn main() {
    let mut s = String::from("Hello");
    s.push_str(", World!");
    println!("{}", s.to_uppercase());
    println!("Length: {}", s.len());
}
```
