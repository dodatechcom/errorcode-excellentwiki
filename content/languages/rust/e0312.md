# E0312: Does Not Outlive

A reference does not live long enough — it will be dropped before it can be used.

## Common Causes

```rust
fn create_ref() -> &str {
    let s = String::from("hello");
    &s  // ERROR: s dropped at end of function
}
```

## How to Fix

1. **Return owned data instead**

```rust
fn create_string() -> String {
    String::from("hello")
}
```

2. **Use static lifetimes**

```rust
fn create_static() -> &'static str {
    "hello"
}
```

3. **Use `'static` or longer lifetimes**

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

## Examples

```rust
fn static_str() -> &'static str {
    "I live forever"
}

fn main() {
    let s = static_str();
    println!("{}", s);

    let owned = String::from("I live in scope");
    let r: &str = &owned;
    println!("{}", r);
}
```
