# E0412: Type Not Found

The compiler cannot find the specified type — check the name and imports.

## Common Causes

```rust
use my_crate::MyType; // ERROR: my_crate not in dependencies
let x: UnknownType = ...; // ERROR: type doesn't exist
```

## How to Fix

1. **Add the dependency to Cargo.toml**

```toml
[dependencies]
serde = { version = "1", features = ["derive"] }
```

2. **Import the correct module**

```rust
use std::collections::HashMap; // not HashSet
use std::io::Read; // not std::io::ReadBuf
```

3. **Use fully qualified paths**

```rust
let map: std::collections::HashMap<String, i32> = std::collections::HashMap::new();
```

## Examples

```rust
use std::collections::{HashMap, HashSet};

fn main() {
    let mut scores: HashMap<&str, i32> = HashMap::new();
    scores.insert("Alice", 95);

    let mut unique: HashSet<i32> = HashSet::new();
    unique.insert(1);
    unique.insert(2);

    println!("Scores: {:?}", scores);
    println!("Unique: {:?}", unique);
}
```
