# E0670: Misformatted Closure

A closure has incorrect syntax — check the closure definition.

## Common Causes

```rust
let f = |x| { x + 1 }; // extra semicolon
let g = |x: i32| -> i32 { x + 1 }; // fine, but mixing syntax
```

## How to Fix

1. **Use correct closure syntax**

```rust
let f = |x: i32| -> i32 { x + 1 };
let g = |x: i32| x + 1; // expression form
```

2. **Single expression closures don't need braces**

```rust
let add = |a, b| a + b;
let is_even = |n| n % 2 == 0;
```

3. **Multi-line closures need braces**

```rust
let process = |x: i32| {
    let doubled = x * 2;
    let halved = x / 2;
    doubled + halved
};
```

## Examples

```rust
fn main() {
    let square = |x| x * x;
    let greet = |name: &str| -> String { format!("Hello, {}!", name) };
    let complex = |nums: Vec<i32>| -> Vec<i32> {
        nums.into_iter().filter(|x| x > &0).map(|x| x * 2).collect()
    };

    println!("{}", square(5));
    println!("{}", greet("World"));
    println!("{:?}", complex(vec![-1, 2, -3, 4]));
}
```
