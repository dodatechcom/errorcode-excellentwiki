# E0624: Private Field

Cannot access a private field of a struct — use public accessors or make it public.

## Common Causes

```rust
mod mymod {
    pub struct User {
        name: String, // private
    }
}

let u = mymod::User { name: "Alice".into() };
println!("{}", u.name); // ERROR: name is private
```

## How to Fix

1. **Make the field public**

```rust
mod mymod {
    pub struct User {
        pub name: String,
    }
}
```

2. **Add a getter method**

```rust
mod mymod {
    pub struct User { name: String }
    impl User {
        pub fn name(&self) -> &str { &self.name }
    }
}
```

3. **Use a builder pattern**

```rust
mod mymod {
    pub struct User { name: String }
    impl User {
        pub fn builder() -> UserBuilder { UserBuilder { name: None } }
    }
    pub struct UserBuilder { name: Option<String> }
    impl UserBuilder {
        pub fn name(mut self, name: &str) -> Self { self.name = Some(name.into()); self }
        pub fn build(self) -> User { User { name: self.name.unwrap_or_default() } }
    }
}
```

## Examples

```rust
mod user_mod {
    pub struct User {
        pub name: String,
        pub age: u32,
    }
    impl User {
        pub fn new(name: &str, age: u32) -> Self {
            User { name: name.into(), age }
        }
    }
}

fn main() {
    let u = user_mod::User::new("Alice", 30);
    println!("{} is {} years old", u.name, u.age);
}
```
