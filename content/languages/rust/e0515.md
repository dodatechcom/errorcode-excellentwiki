# E0515: Cannot Return Reference

Cannot return a reference to a value owned by the local function.

## Common Causes

```rust
fn create_ref() -> &String {
    let s = String::from("hello");
    &s  // ERROR: s dropped here
}
```

## How to Fix

1. **Return owned data**

```rust
fn create_string() -> String {
    String::from("hello")
}
```

2. **Return a reference to input**

```rust
fn first_char(s: &str) -> Option<&str> {
    s.get(..1)
}
```

3. **Use Cow for conditional ownership**

```rust
use std::borrow::Cow;

fn greet(name: &str) -> Cow<'_, str> {
    if name.is_empty() {
        Cow::Borrowed("stranger")
    } else {
        Cow::Owned(format!("Hello, {}!", name))
    }
}
```

## Examples

```rust
fn main() {
    let greeting = String::from("Hello");
    let r: &str = &greeting; // reference to owned string
    println!("{}", r);
    // greeting still valid here
}
```
