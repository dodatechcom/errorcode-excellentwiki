# E0432: Unresolved Import

The compiler cannot find the imported item — check module paths and feature flags.

## Common Causes

```rust
use std::collections::HashMapt; // Typo in name
use crate::mymodule::Thing; // mymodule doesn't exist
```

## How to Fix

1. **Verify the module path**

```rust
use std::collections::HashMap;
```

2. **Check visibility**

```rust
mod inner {
    pub struct PublicThing;
}
use inner::PublicThing; // must be pub
```

3. **Use glob imports carefully**

```rust
use std::collections::*;
```

## Examples

```rust
use std::collections::{HashMap, BTreeMap, HashSet};

fn main() {
    let mut map = HashMap::new();
    map.insert("key", "value");
    println!("{:?}", map);
}
```
