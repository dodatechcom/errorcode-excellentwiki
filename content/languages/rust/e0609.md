# E0609: No Field

The type does not have the requested field — check struct definitions.

## Common Causes

```rust
struct User { name: String }
let u = User { name: "Alice".into() };
println!("{}", u.age); // ERROR: no field 'age'
```

## How to Fix

1. **Check field names**

```rust
struct User { name: String, age: u32 }
let u = User { name: "Alice".into(), age: 30 };
println!("{}", u.age);
```

2. **Use tuple indexing for tuples**

```rust
let t = (1, 2, 3);
println!("{}", t.0);
```

3. **Use match for enum variants**

```rust
enum Shape { Circle(f64), Square(f64) }
match shape {
    Shape::Circle(r) => println!("radius: {}", r),
    Shape::Square(s) => println!("side: {}", s),
}
```

## Examples

```rust
struct Point { x: f64, y: f64 }

fn main() {
    let p = Point { x: 1.0, y: 2.0 };
    println!("x={}, y={}", p.x, p.y);
}
```
