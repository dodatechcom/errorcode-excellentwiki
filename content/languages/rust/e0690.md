# E0690: Cannot Contain Self-referential Type

A struct contains a field of its own type, creating infinite recursion.

## Common Causes

```rust
struct Node {
    child: Node, // ERROR: infinite size
}
```

## How to Fix

1. **Use Box for recursive types**

```rust
struct Node {
    value: i32,
    child: Option<Box<Node>>,
}
```

2. **Use references with lifetimes**

```rust
struct Node<'a> {
    value: i32,
    child: Option<&'a Node<'a>>,
}
```

3. **Use indices instead of pointers**

```rust
struct Tree {
    nodes: Vec<Node>,
}

struct Node {
    value: i32,
    children: Vec<usize>,
}
```

## Examples

```rust
enum List<T> {
    Cons(T, Box<List<T>>),
    Nil,
}

fn main() {
    use List::{Cons, Nil};
    let list = Cons(1, Box::new(Cons(2, Box::new(Cons(3, Box::new(Nil))))));
}
```
