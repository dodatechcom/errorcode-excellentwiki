# E0463: Binary Not Found

The binary target specified does not exist — check Cargo.toml configuration.

## Common Causes

```rust
// Running: cargo run --bin wrong_name
// [[bin]] in Cargo.toml points to non-existent file
```

## How to Fix

1. **Check Cargo.toml bin targets**

```toml
[[bin]]
name = "my_app"
path = "src/main.rs"
```

2. **Use default binary**

```rust
// Just use cargo run without --bin
```

3. **List available binaries**

```bash
cargo metadata --no-deps --format-version 1 | jq '.packages[0].targets[] | select(.kind[0]=="bin")'
```

## Examples

```toml
[package]
name = "my_app"
version = "0.1.0"
edition = "2021"

[[bin]]
name = "server"
path = "src/bin/server.rs"

[[bin]]
name = "cli"
path = "src/bin/cli.rs"
```

```bash
cargo run --bin server
cargo run --bin cli
```
