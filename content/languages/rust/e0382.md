# E0382: Use of Moved Value

A variable was used after its value was moved — clone or copy the value.

## Common Causes

```rust
let s1 = String::from("hello");
let s2 = s1;       // s1 moved to s2
println!("{}", s1); // ERROR: value used after move

let v = vec![1, 2, 3];
let v2 = v;
v.push(4); // ERROR: v moved
```

## How to Fix

1. **Clone the value**

```rust
let s1 = String::from("hello");
let s2 = s1.clone();
println!("{} and {}", s1, s2);
```

2. **Use Copy types**

```rust
let x: i32 = 42;
let y = x;
println!("{} and {}", x, y); // i32 implements Copy
```

3. **Borrow instead of move**

```rust
fn print_str(s: &str) { println!("{}", s); }

let s1 = String::from("hello");
print_str(&s1); // borrow, don't move
println!("{}", s1); // still valid
```

## Examples

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1.clone();
    println!("s1: {}, s2: {}", s1, s2);

    let v = vec![1, 2, 3];
    let len = v.len(); // borrow
    println!("vec has {} elements: {:?}", len, v);
}
```
