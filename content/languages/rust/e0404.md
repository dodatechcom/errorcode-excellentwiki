# E0404: Not a Trait

Found a type where a trait was expected — this item is not a trait.

## Common Causes

```rust
struct MyStruct;
fn needs_trait<T: MyStruct>() {} // ERROR: MyStruct is not a trait

impl MyStruct for String {} // ERROR: can only implement traits
```

## How to Fix

1. **Use trait bounds, not struct bounds**

```rust
struct MyStruct;

trait MyTrait {
    fn do_thing(&self);
}

impl MyTrait for MyStruct {
    fn do_thing(&self) { println!("doing thing"); }
}

fn needs_trait<T: MyTrait>(item: &T) {
    item.do_thing();
}
```

2. **Ensure you're implementing a trait**

```rust
trait Greet {
    fn hello(&self) -> String;
}

impl Greet for &str {
    fn hello(&self) -> String {
        format!("Hello, {}!", self)
    }
}
```

3. **Check imports**

```rust
use std::fmt::Display; // ensure trait is in scope
```

## Examples

```rust
trait Printable {
    fn print(&self);
}

struct Article {
    title: String,
    content: String,
}

impl Printable for Article {
    fn print(&self) {
        println!("{}: {}", self.title, self.content);
    }
}

fn main() {
    let art = Article {
        title: "Rust".into(),
        content: "Rust is great".into(),
    };
    art.print();
}
```
