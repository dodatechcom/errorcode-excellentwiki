# E0597: Does Not Outlive

A temporary value does not live long enough — it will be dropped.

## Common Causes

```rust
let r;
{
    let x = 5;
    r = &x; // ERROR: x dropped here
}
println!("{}", r);
```

## How to Fix

1. **Extend the value's lifetime**

```rust
let x = 5;
let r = &x; // x lives long enough
println!("{}", r);
```

2. **Use static values**

```rust
let r: &'static str = "hello";
```

3. **Clone into a new binding**

```rust
let r;
let x = String::from("hello");
r = x.clone();
println!("{}", r);
```

## Examples

```rust
fn main() {
    let data = String::from("persistent");
    let reference = &data;
    println!("{}", reference);
    // data still alive here
}
```
