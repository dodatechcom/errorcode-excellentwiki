# E0618: Expected Closure

Called a value that is not a function — it may be a struct or enum variant.

## Common Causes

```rust
struct MyStruct { value: i32 }
let s = MyStruct { value: 42 };
s(); // ERROR: MyStruct is not callable
```

## How to Fix

1. **Call the correct value**

```rust
let add = |a, b| a + b;
println!("{}", add(1, 2));
```

2. **Use Fn traits**

```rust
fn apply<F: Fn(i32) -> i32>(f: F, x: i32) -> i32 {
    f(x)
}
```

3. **Check naming**

```rust
fn my_function() -> i32 { 42 }
println!("{}", my_function()); // not my_function
```

## Examples

```rust
fn main() {
    let multiply = |a: i32, b: i32| -> i32 { a * b };
    println!("3 * 4 = {}", multiply(3, 4));

    let numbers = vec![1, 2, 3, 4, 5];
    let sum: i32 = numbers.iter().fold(0, |acc, &x| acc + x);
    println!("Sum: {}", sum);
}
```
