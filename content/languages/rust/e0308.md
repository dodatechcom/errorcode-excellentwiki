# E0308: Mismatched Types

The compiler found a value of one type where another is expected.

## Common Causes

```rust
let x: i32 = "hello";     // ERROR: expected i32, got &str
let y: bool = 42;          // ERROR: expected bool, got integer
let v: Vec<i32> = vec!["a", "b"]; // ERROR: wrong element type
```

## How to Fix

1. **Use explicit type conversion**

```rust
let x: i32 = "42".parse().unwrap();
let y: f64 = 42 as f64;
let z: String = 42.to_string();
```

2. **Use the correct type**

```rust
let x: i32 = 42;
let y: f64 = 42.0;
let v: Vec<&str> = vec!["a", "b"];
```

3. **Use turbofish for ambiguous types**

```rust
let v = "42".parse::<i32>().unwrap();
let v = vec![1, 2, 3].into_iter().collect::<Vec<_>>();
```

## Examples

```rust
fn double(x: i32) -> i32 { x * 2 }

fn main() {
    let s = "100";
    let n: i32 = s.parse().unwrap();
    println!("{} * 2 = {}", n, double(n));

    let nums: Vec<i32> = (1..=5).collect();
    let sum: i32 = nums.iter().sum();
    println!("Sum: {}", sum);
}
```
