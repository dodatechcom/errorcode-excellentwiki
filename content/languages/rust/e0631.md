# E0631: Type Mismatch in Closure

The closure return type or parameter type does not match expected.

## Common Causes

```rust
let f = |x: i32| -> &str { "hello" }; // OK
let g: Box<dyn Fn(i32) -> i32> = Box::new(|x| x as f64); // ERROR
```

## How to Fix

1. **Ensure consistent types**

```rust
let f: Box<dyn Fn(i32) -> i32> = Box::new(|x| x * 2);
```

2. **Use explicit type annotations**

```rust
let f = |x: i32| -> i32 { x + 1 };
```

3. **Let inference work**

```rust
let v = vec![1, 2, 3];
let mapped: Vec<i32> = v.into_iter().map(|x| x * 2).collect();
```

## Examples

```rust
fn main() {
    let numbers = vec![1, 2, 3, 4, 5];
    let evens: Vec<i32> = numbers.into_iter()
        .filter(|x| x % 2 == 0)
        .collect();
    println!("Evens: {:?}", evens);

    let strs: Vec<String> = vec![1, 2, 3].into_iter()
        .map(|x| x.to_string())
        .collect();
    println!("Strings: {:?}", strs);
}
```
