# E0507: Cannot Move Out

Cannot move a value out of a borrowed context — use references or clones.

## Common Causes

```rust
let v = vec![1, 2, 3];
let first = v[0]; // ERROR: cannot move out of borrowed Vec
```

## How to Fix

1. **Borrow the value**

```rust
let v = vec![1, 2, 3];
let first = &v[0]; // borrow, don't move
```

2. **Clone the value**

```rust
let v = vec![String::from("hello")];
let first = v[0].clone(); // clone to move
```

3. **Use Option::take or swap_remove**

```rust
let mut v = vec![1, 2, 3];
let first = v.remove(0); // moves and shifts
```

## Examples

```rust
fn main() {
    let words = vec![String::from("hello"), String::from("world")];
    let first: &str = &words[0]; // borrow
    println!("First: {}", first);
    println!("Still valid: {:?}", words);
}
```
