# E0619: Type Mismatch in Cast

The cast operation is invalid for the given types.

## Common Causes

```rust
let x: String = 42 as String; // ERROR: cannot cast
let y: bool = 42 as bool; // ERROR: invalid cast
```

## How to Fix

1. **Use From/Into traits**

```rust
let x: i32 = 42;
let y: f64 = x as f64; // numeric cast OK
```

2. **Use proper conversion**

```rust
let x: i32 = 42;
let s: String = x.to_string(); // not as String
```

3. **Use safe casting**

```rust
let x: u8 = 42;
let y: u16 = x as u16; // safe widening
```

## Examples

```rust
fn main() {
    let x: i32 = 42;
    let y: f64 = x as f64;
    let z: i64 = x as i64;
    println!("i32={}, f64={}, i64={}", x, y, z);
}
```
