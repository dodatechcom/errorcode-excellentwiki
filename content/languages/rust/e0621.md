# E0621: Type Mismatch in Closure Arg

The closure argument type does not match the expected type.

## Common Causes

```rust
let v = vec![String::from("hello")];
v.iter().for_each(|x: &str| println!("{}", x));
// ERROR: x is &String, not &str (though deref coercion may help)
```

## How to Fix

1. **Use correct types**

```rust
let v = vec![String::from("hello")];
v.iter().for_each(|x: &String| println!("{}", x));
```

2. **Let type inference work**

```rust
v.iter().for_each(|x| println!("{}", x));
```

3. **Use as_str() or as_ref()**

```rust
v.iter().for_each(|x| println!("{}", x.as_str()));
```

## Examples

```rust
fn main() {
    let numbers = vec![1, 2, 3, 4, 5];
    numbers.iter().for_each(|n| {
        println!("Value: {}", n);
    });

    let doubled: Vec<i32> = numbers.iter().map(|&x| x * 2).collect();
    println!("Doubled: {:?}", doubled);
}
```
