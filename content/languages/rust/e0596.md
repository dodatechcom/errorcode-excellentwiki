# E0596: Cannot Borrow Mutably

Cannot borrow a value as mutable — the value is not declared as `mut`.

## Common Causes

```rust
let v = vec![1, 2, 3];
v.push(4); // ERROR: cannot borrow as mutable
```

## How to Fix

1. **Declare variable as mutable**

```rust
let mut v = vec![1, 2, 3];
v.push(4); // OK
```

2. **Use interior mutability**

```rust
use std::cell::RefCell;

let data = RefCell::new(vec![1, 2, 3]);
data.borrow_mut().push(4);
```

3. **Use Mutex for thread safety**

```rust
use std::sync::Mutex;

let data = Mutex::new(vec![1, 2, 3]);
data.lock().unwrap().push(4);
```

## Examples

```rust
fn main() {
    let mut numbers = vec![1, 2, 3];
    numbers.push(4);
    numbers.push(5);
    println!("{:?}", numbers);

    let mut name = String::from("Hello");
    name.push_str(", World!");
    println!("{}", name);
}
```
