# E0614: Cannot Dereference

Cannot dereference this type — it doesn't implement Deref or isn't a reference.

## Common Causes

```rust
let x = 5;
let y = *x; // ERROR: x is not a reference
```

## How to Fix

1. **Use references**

```rust
let x = &5;
let y = *x; // OK: dereference
```

2. **Implement Deref for custom types**

```rust
use std::ops::Deref;

struct MyBox<T>(T);

impl<T> Deref for MyBox<T> {
    type Target = T;
    fn deref(&self) -> &T { &self.0 }
}
```

3. **Use .as_ref() or .as_deref()**

```rust
let s: Option<String> = Some("hello".into());
let r: Option<&str> = s.as_deref();
```

## Examples

```rust
fn main() {
    let s = String::from("hello");
    let r: &str = &s; // Deref coercion: &String -> &str
    println!("{}", r);

    let n = 42;
    let r = &n;
    println!("{}", *r); // explicit deref
}
```
