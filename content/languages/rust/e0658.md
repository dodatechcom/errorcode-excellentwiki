# E0658: Use of Unstable Feature

A nightly-only feature was used on stable Rust — enable nightly or avoid the feature.

## Common Causes

```rust
#![feature(generators)] // ERROR: unstable
let gen = || { yield 1; yield 2; };
```

## How to Fix

1. **Use stable alternatives**

```rust
// Instead of generators, use Iterator impl
struct Counter { count: u32 }
impl Iterator for Counter {
    type Item = u32;
    fn next(&mut self) -> Option<u32> {
        if self.count < 5 { self.count += 1; Some(self.count) } else { None }
    }
}
```

2. **Use nightly toolchain**

```bash
rustup toolchain install nightly
rustup override set nightly
```

3. **Use feature-gated crates**

```toml
[dependencies]
async-stream = "0.2"  # stable async streams
```

## Examples

```rust
// Use stable Rust features
use std::collections::HashMap;
use std::sync::{Arc, Mutex};
use tokio::sync::mpsc;

#[tokio::main]
async fn main() {
    let (tx, mut rx) = mpsc::channel(32);
    tokio::spawn(async move { tx.send("hello").await.unwrap(); });
    println!("{:?}", rx.recv().await);
}
```
