# E0433: Failed to Resolve

The compiler cannot resolve a path — module or crate not found.

## Common Causes

```rust
my_crate::function(); // my_crate not in Cargo.toml
std::collections::new::HashMap(); // wrong path
```

## How to Fix

1. **Add crate dependency**

```toml
[dependencies]
serde = "1"
```

```rust
use serde::Serialize;
```

2. **Use correct module paths**

```rust
use std::collections::HashMap;
let map: HashMap<String, i32> = HashMap::new();
```

3. **Check feature flags**

```toml
[dependencies]
tokio = { version = "1", features = ["full"] }
```

## Examples

```rust
use std::collections::HashMap;
use std::fmt;

struct Config {
    settings: HashMap<String, String>,
}

impl fmt::Display for Config {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        for (k, v) in &self.settings {
            writeln!(f, "{}: {}", k, v)?;
        }
        Ok(())
    }
}

fn main() {
    let mut config = Config { settings: HashMap::new() };
    config.settings.insert("host".into(), "localhost".into());
    println!("{}", config);
}
```
