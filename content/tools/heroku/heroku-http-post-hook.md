Heroku HTTP Post Hook Error can prevent your application from working correctly.

## Common Causes

- Hook not firing
- URL unreachable
- Payload invalid

## How to Fix

### Configure Hook

1. Go to app settings
2. Add deploy hook
3. Enter URL
