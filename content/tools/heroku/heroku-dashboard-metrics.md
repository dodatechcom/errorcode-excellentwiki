Heroku Dashboard Metrics Error can prevent your application from working correctly.

## Common Causes

- Metrics not loading
- Data outdated
- Chart error

## How to Fix

### Check Metrics

1. Go to Heroku dashboard
2. Select app
3. Go to Metrics tab
