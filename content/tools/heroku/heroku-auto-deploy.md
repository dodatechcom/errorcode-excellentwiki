Heroku Auto Deploy Error can prevent your application from working correctly.

## Common Causes

- Auto deploy not configured
- Deploy failed
- Branch not matched

## How to Fix

### Configure Auto Deploy

1. Go to Pipeline settings
2. Enable auto deploy
