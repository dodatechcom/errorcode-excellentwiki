Vercel Bitbucket Install Error can prevent your application from working correctly.

## Common Causes

- Bitbucket not connected
- Permission denied
- Workspace access revoked

## How to Fix

### Connect Bitbucket

1. Go to Vercel Integrations
2. Find Bitbucket
3. Click Connect
