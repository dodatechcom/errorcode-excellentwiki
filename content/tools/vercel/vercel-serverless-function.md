Vercel Serverless Function Error can prevent your application from working correctly.

## Common Causes

- Function crashes at runtime
- Import errors
- Environment variable missing
- Function exceeds size limit

## How to Fix

### Check Logs

```bash
npx vercel logs
```

### Test Locally

```bash
npx vercel dev
```
