Vercel Production Deployment Error can prevent your application from working correctly.

## Common Causes

- Build failed
- Configuration error
- Environment variable missing
- Domain not configured

## How to Fix

### Deploy

```bash
npx vercel --prod
```

### Check Status

```bash
npx vercel ls
```
