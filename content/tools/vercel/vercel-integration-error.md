Vercel Integration Error can prevent your application from working correctly.

## Common Causes

- Integration not configured
- API key invalid
- Webhook not receiving events
- Service unavailable

## How to Fix

### Configure Integration

1. Go to Project Settings > Integrations
2. Add integration
3. Configure credentials
