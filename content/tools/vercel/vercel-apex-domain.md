Vercel Apex Domain Error can prevent your application from working correctly.

## Common Causes

- Apex domain not configured
- A record missing
- DNS configuration wrong

## How to Fix

### Configure Apex Domain

1. Go to Project Settings > Domains
2. Add apex domain
3. Add A record: `76.76.21.21`
