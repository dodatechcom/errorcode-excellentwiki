Vercel GitHub Install Error can prevent your application from working correctly.

## Common Causes

- GitHub App not installed
- Permission denied
- Repository access revoked

## How to Fix

### Install GitHub App

1. Go to Vercel Integrations
2. Find GitHub
3. Click Install

### Grant Access

Select repositories to grant access.
