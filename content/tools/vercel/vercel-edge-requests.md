Vercel Edge Requests Error can prevent your application from working correctly.

## Common Causes

- Edge request limit exceeded
- Function too slow
- Edge location unavailable
- Function size exceeded

## How to Fix

### Check Usage

1. Go to Team Settings > Usage
2. Review edge requests
