Vercel Third-Party Integration Error can prevent your application from working correctly.

## Common Causes

- Service not connected
- Credentials expired
- API version mismatch
- Service rate limited

## How to Fix

### Reconnect Integration

1. Go to Project Settings > Integrations
2. Click on integration
3. Reconnect or update credentials
