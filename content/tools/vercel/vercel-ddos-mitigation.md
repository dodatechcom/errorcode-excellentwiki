Vercel DDoS Mitigation Error can prevent your application from working correctly.

## Common Causes

- DDoS attack detected
- Legitimate traffic blocked
- Rate limiting too aggressive
- IP blocked incorrectly

## How to Fix

### Check Status

1. Go to Project Settings > Security
2. Review DDoS protection settings
