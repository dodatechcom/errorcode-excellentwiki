Vercel Git Integration Error can prevent your application from working correctly.

## Common Causes

- Repository not connected
- Permission denied
- Branch not configured
- Webhook not set

## How to Fix

### Connect Repository

1. Go to Project Settings > Git
2. Click Import
3. Select repository
