Vercel Marketplace Add-on Error can prevent your application from working correctly.

## Common Causes

- Add-on not provisioned
- Plan limit reached
- Add-on configuration invalid
- Service unavailable

## How to Fix

### Provision Add-on

1. Go to Project Settings > Storage
2. Add marketplace add-on
