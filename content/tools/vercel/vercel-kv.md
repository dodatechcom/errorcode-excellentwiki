Vercel KV Error can prevent your application from working correctly.

## Common Causes

- KV namespace not configured
- Connection failed
- Key not found
- Value too large

## How to Fix

### Configure KV

1. Go to Vercel dashboard
2. Create KV store
3. Link to project
