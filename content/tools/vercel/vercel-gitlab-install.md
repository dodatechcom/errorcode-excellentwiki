Vercel GitLab Install Error can prevent your application from working correctly.

## Common Causes

- GitLab not connected
- Permission denied
- Access token expired

## How to Fix

### Connect GitLab

1. Go to Vercel Integrations
2. Find GitLab
3. Click Connect

### Authorize

Grant access to your GitLab account.
