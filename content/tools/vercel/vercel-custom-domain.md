Vercel Custom Domain Error can prevent your application from working correctly.

## Common Causes

- Domain already in use
- DNS configuration wrong
- SSL certificate pending
- Domain limit reached

## How to Fix

### Add Domain

1. Go to Project Settings > Domains
2. Click Add
3. Enter domain name

### Configure DNS

Add CNAME or A record pointing to Vercel.
