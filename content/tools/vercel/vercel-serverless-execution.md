Vercel Serverless Execution Error can prevent your application from working correctly.

## Common Causes

- Execution limit exceeded
- Function timeout
- Cold start too slow
- Function crashed

## How to Fix

### Check Usage

1. Go to Team Settings > Usage
2. Review serverless execution

### Optimize

Reduce function execution time and frequency.
