Vercel Bandwidth Limit Error can prevent your application from working correctly.

## Common Causes

- Bandwidth limit exceeded
- Large file downloads
- High traffic volume
- CDN not configured

## How to Fix

### Check Usage

1. Go to Team Settings > Usage
2. Review bandwidth

### Optimize

- Enable CDN caching
- Compress responses
- Use image optimization
