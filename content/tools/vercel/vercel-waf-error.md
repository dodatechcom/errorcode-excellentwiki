Vercel WAF Error can prevent your application from working correctly.

## Common Causes

- WAF blocking legitimate traffic
- Rule too aggressive
- Custom rule syntax error
- WAF not enabled

## How to Fix

### Configure WAF

1. Go to Project Settings > Security
2. Configure WAF rules
