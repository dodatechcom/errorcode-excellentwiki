Cloudflare Gateway Policy Error can prevent your application from working correctly.

## Common Causes

- Policy not enabled
- Rules too broad
- Identity not configured
- Policy order conflicts

## How to Fix

### Create Policy

1. Go to Zero Trust > Gateway > Policies
2. Select policy type (DNS, HTTP, Network)
3. Configure rules
