Cloudflare Turnstile Site Key Error can prevent your application from working correctly.

## Common Causes

- Site key copied incorrectly
- Site key does not match domain
- Site key from wrong environment
- Site key was rotated

## How to Fix

### Verify Key

1. Go to Cloudflare dashboard
2. Navigate to Turnstile
3. Copy correct site key
