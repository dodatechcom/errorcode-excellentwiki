Cloudflare Mirage Error can prevent your application from working correctly.

## Common Causes

- Mirage disabled
- Images not proxied
- Mobile detection issues
- Image format incompatibility

## How to Fix

### Enable Mirage

1. Go to Speed > Optimization
2. Enable Mirage
