Cloudflare Auto Minify Error can prevent your application from working correctly.

## Common Causes

- Minification breaks code
- Syntax error after minification
- Minify setting too aggressive
- Files not served through Cloudflare

## How to Fix

### Enable

1. Go to Speed > Optimization
2. Enable Auto Minify for CSS, JavaScript, HTML
