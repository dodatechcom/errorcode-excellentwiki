Cloudflare Speed Test Error can prevent your application from working correctly.

## Common Causes

- Site not accessible
- Firewall blocking test
- Slow origin response
- Large page size

## How to Fix

### Run Test

1. Go to Speed > Overview
2. Click Run Test
3. Review results
