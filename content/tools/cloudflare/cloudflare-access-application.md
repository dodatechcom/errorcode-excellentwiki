Cloudflare Access Application Error can prevent your application from working correctly.

## Common Causes

- Application domain not verified
- Session duration misconfigured
- Application type incorrect
- CORS settings wrong

## How to Fix

### Create Application

1. Go to Zero Trust > Access > Applications
2. Click Add an application
3. Configure domain and policies
