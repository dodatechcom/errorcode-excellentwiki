Netlify Branch Deploy Error can prevent your application from working correctly.

## Common Causes

- Branch deploy not configured
- Deploy failed
- Branch not found
- Deploy limit reached

## How to Fix

### Configure Branch Deploy

1. Go to Site Settings > Build & deploy > Branches
2. Add branch
