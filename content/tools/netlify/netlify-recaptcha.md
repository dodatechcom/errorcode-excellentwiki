Netlify reCAPTCHA Error can prevent your application from working correctly.

## Common Causes

- reCAPTCHA not configured
- Site key invalid
- Verification failing
- reCAPTCHA not showing

## How to Fix

### Enable reCAPTCHA

1. Go to Forms > Spam filter
2. Enable reCAPTCHA
3. Add site key
