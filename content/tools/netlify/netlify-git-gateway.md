Netlify Git Gateway Error can prevent your application from working correctly.

## Common Causes

- Git Gateway not configured
- Access token invalid
- Repository not connected
- Gateway URL wrong

## How to Fix

### Configure Git Gateway

1. Go to Site Settings > Identity > Git Gateway
2. Connect to GitHub/GitLab
