Netlify Bandwidth Report Error can prevent your application from working correctly.

## Common Causes

- Bandwidth not tracked
- Usage exceeds limit
- Report not accurate

## How to Fix

### Check Usage

1. Go to Team Settings > Billing
2. View bandwidth usage
