Netlify Domain Alias Error can prevent your application from working correctly.

## Common Causes

- Alias not configured
- DNS not pointing to Netlify
- SSL not provisioned for alias

## How to Fix

### Add Domain Alias

1. Go to Domain Settings
2. Click Add alias
3. Enter domain
