Netlify Site Collaborator Error can prevent your application from working correctly.

## Common Causes

- Collaborator not added
- Permission insufficient
- Collaborator limit reached

## How to Fix

### Add Collaborator

1. Go to Site Settings > Access control
2. Add collaborator
