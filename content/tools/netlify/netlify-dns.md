Netlify DNS Error can prevent your application from working correctly.

## Common Causes

- DNS not configured
- Nameservers not updated
- Record missing
- DNS propagation delay

## How to Fix

### Configure DNS

1. Go to Domain Settings
2. Add custom domain
3. Configure DNS records
