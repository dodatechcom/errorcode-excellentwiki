Netlify Git Gateway Error can prevent your application from working correctly.

## Common Causes

- Gateway not connected
- Token invalid
- Repository access denied
- Branch not found

## How to Fix

### Reconnect Gateway

1. Go to Site Settings > Identity > Git Gateway
2. Reconnect
