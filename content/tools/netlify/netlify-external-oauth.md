Netlify External OAuth Error can prevent your application from working correctly.

## Common Causes

- OAuth provider not configured
- Client ID invalid
- Callback URL wrong
- Provider unavailable

## How to Fix

### Configure OAuth

1. Go to Identity > External providers
2. Enable provider
3. Enter Client ID and Secret
