Netlify Custom Domain Error can prevent your application from working correctly.

## Common Causes

- Domain already in use
- DNS not configured
- SSL certificate pending
- Domain limit reached

## How to Fix

### Add Domain

1. Go to Domain Settings
2. Click Add custom domain
3. Enter domain name
