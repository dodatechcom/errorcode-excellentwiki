Netlify CI Build Error can prevent your application from working correctly.

## Common Causes

- CI environment not configured
- Test command failed
- Build not triggered
- Webhook not configured

## How to Fix

### Configure CI

1. Go to Site Settings > Build & deploy
2. Configure CI settings
