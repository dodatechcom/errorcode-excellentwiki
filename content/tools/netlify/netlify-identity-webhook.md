Netlify Identity Webhook Error can prevent your application from working correctly.

## Common Causes

- Webhook not configured
- URL incorrect
- Payload invalid
- Webhook not triggered

## How to Fix

### Configure Webhook

1. Go to Identity > Settings > Webhooks
2. Add webhook URL
