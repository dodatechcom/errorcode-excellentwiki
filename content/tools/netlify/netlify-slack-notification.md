Netlify Slack Notification Error can prevent your application from working correctly.

## Common Causes

- Slack not connected
- Channel not specified
- Webhook URL invalid
- Permission denied

## How to Fix

### Connect Slack

1. Go to Integrations
2. Add Slack
3. Authorize
