Netlify Draft PR Error can prevent your application from working correctly.

## Common Causes

- Draft PR not previewed
- Preview not generated
- PR not connected

## How to Fix

### Enable Draft Preview

1. Go to Site Settings > Build & deploy
2. Enable draft PR previews
