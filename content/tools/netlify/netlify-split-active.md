Netlify Split Test Active Error can prevent your application from working correctly.

## Common Causes

- Test already active
- Cannot modify active test
- Traffic allocation wrong

## How to Fix

### Manage Active Test

1. Go to Split Testing
2. View active tests
3. Pause or modify
