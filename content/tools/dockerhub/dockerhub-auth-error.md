# Docker Hub Authentication Error

Docker Hub authentication errors occur when login, token, or credential management fails.

## Why This Happens

- Invalid credentials
- Token expired
- 2FA required
- Account locked

## Common Error Messages

- `auth_invalid_error`
- `auth_token_error`
- `auth_2fa_error`
- `auth_locked_error`

## How to Fix It

### Solution 1: Check credentials

Verify your username and password.

### Solution 2: Use access tokens

Create access tokens in Docker Hub settings.

### Solution 3: Handle 2FA

Complete two-factor authentication.


## Common Scenarios

- **Invalid credentials:** Reset your password if needed.
- **Token expired:** Generate a new access token.

## Prevent It

- Use access tokens
- Enable 2FA
- Rotate credentials regularly
