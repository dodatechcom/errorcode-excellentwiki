DockerHub Scoped Token Error can prevent your application from working correctly.

## Common Causes

- Token scope insufficient
- Read-only token
- Write access needed

## How to Fix

### Create Scoped Token

1. Go to Account Settings > Security
2. Create token with correct scopes
