DockerHub Repository Public Error can prevent your application from working correctly.

## Common Causes

- Cannot make private
- Public exposure risk
- Repo should be private

## How to Fix

### Change Visibility

1. Go to Docker Hub
2. Select repository
3. Change visibility settings
