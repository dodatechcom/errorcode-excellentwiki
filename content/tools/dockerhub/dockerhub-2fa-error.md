# Docker Hub Two-Factor Auth Error

Docker Hub 2FA errors occur when two-factor authentication setup or login fails.

## Why This Happens

- 2FA not enabled
- Code invalid
- Backup code lost
- Recovery failed

## Common Error Messages

- `2fa_not_enabled_error`
- `2fa_code_error`
- `2fa_backup_error`
- `2fa_recovery_error`

## How to Fix It

### Solution 1: Enable 2FA

Set up two-factor authentication in Docker Hub settings.

### Solution 2: Use correct code

Enter the 2FA code from your authenticator app.

### Solution 3: Use backup codes

Use backup codes if you lost access to your authenticator.


## Common Scenarios

- **2FA not enabled:** Enable 2FA in security settings.
- **Code invalid:** Check the time on your authenticator device.

## Prevent It

- Enable 2FA for security
- Store backup codes safely
- Use authenticator app
