# CircleCI Organization Error

CircleCI organization errors occur when org-level settings, permissions, or billing issues prevent pipeline execution.

## Why This Happens

- Org not found
- Permission denied
- Billing issue
- SSO not configured

## Common Error Messages

- `org_not_found`
- `permission_denied`
- `billing_error`
- `sso_error`

## How to Fix It

### Solution 1: Check org membership

Verify you're a member of the organization.

### Solution 2: Fix permissions

Request appropriate permissions from org admin.

### Solution 3: Check billing status

Verify the organization's billing is active.


## Common Scenarios

- **Permission denied:** Contact org admin for access.
- **Billing expired:** Update payment method.

## Prevent It

- Verify org membership
- Check billing status
- Configure SSO
