# Grafana Grafana Cloud Error

Grafana Cloud errors occur when cloud-hosted Grafana instances encounter connectivity or configuration issues.

## Why This Happens

- Cloud connection failed
- API key invalid
- Quota exceeded
- Region mismatch

## Common Error Messages

- `cloud_connection_error`
- `cloud_auth_error`
- `cloud_quota_error`
- `cloud_region_error`

## How to Fix It

### Solution 1: Check cloud status

Monitor the Grafana Cloud status page.

### Solution 2: Verify API keys

Check cloud API keys in the cloud console.

### Solution 3: Monitor quotas

Track usage against cloud quotas.


## Common Scenarios

- **Connection failed:** Check internet connectivity.
- **Quota exceeded:** Upgrade plan or optimize usage.

## Prevent It

- Monitor cloud status
- Track usage
- Set up alerts
